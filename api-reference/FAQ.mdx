## 1. What is the WATO API?

The WATO API is a set of endpoints that allow developers to integrate WATO's features into their own applications. It enables you to manage WhatsApp communities, groups, and streamline communication with customers on WhatsApp.

## 2. How do I get started with the WATO API?

To get started with the WATO API, you'll need to sign up for an API key. If you're already a subscribed user, you can obtain your API key by using the [authorize_user API](/api-reference/endpoint/authorize_user) endpoint.

<Note>
  If you're not yet a subscribed user, please contact our sales team at [info@wato.ai](mailto:info@wato.ai). We'll get in touch with you to provide a product demo and discuss commercials.
</Note>

## 3. What authentication method does the WATO API use?

The WATO API uses Bearer token authentication. You'll need to include your API key in the authorization header of your requests to authenticate.

## 4. Where can I find the documentation for the WATO API endpoints?

You can find the documentation for the WATO API endpoints on our [API reference page](/api-reference).

## 5. Does the WATO API support webhooks?

Yes, the WATO API supports webhooks. You can use webhooks to receive real-time notifications about events that occur in your WhatsApp communities and groups.

If you have any further questions or need assistance with the WATO API, you can contact our support team at [support@wato.ai](mailto:support@wato.ai). We're here to help!
